# Kedro's spaceflights starter in Ordeq

This project contains the Kedro spaceflights starter, migrated to Ordeq.
It's meant for users that are familiar with Kedro, and looking to migrate to Ordeq.
It is based on [this][kedro-starter].

To run the pipeline, run:

```shell
uv run -m kedro_spaceflights run kedro_spaceflights.pipeline
```

[kedro-starter]: https://github.com/kedro-org/kedro-starters/tree/main/spaceflights-pandas
